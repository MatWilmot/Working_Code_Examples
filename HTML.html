Text Formatting

Bold Text

You can make text bold by adding a pair of <strong></strong> tags around the desired text.

Example:

He was this close to scoring a goal!

Itallics

You can make text itallic by adding a pair of <em></em> tags around the desired text.

Example:

He was this close to scoring a goal!

Underline

You can make text underlined by adding a pair of <u></u> tags around the desired text.

Example:

He was this close to scoring a goal!

Strikethrough

You can make text have a strikethrough by adding a pair of <s></s> tags around the desired text.

He was this close wasn't even close to scoring a goal!

Headers

Header tags can be used to create text in various preset sizes, and come with a number of formatting options already applied to them. Use tags <h1></h1> to <h6></h6> to get the size you want.

Example:

This is h1 text

This is h2 text

This is h3 text

This is h4 text

This is h5 text
This is h6 text

Anchor Tags

Anchor tags allow you to create links to things like external web pages, or even elements within the current page. The nav bar on the left uses anchor tags to navigate up and down the current page, bringing you to whichever header you click on. Anchor tags - <a></a> - always contain an 'href' parameter that determines the destination of the element.

Example:

<a href="https://www.youtube.com">YouTube</a>

This code would create this: YouTube


<a href="#secion-header">Section Title</a>

This code would create a link that would take you to whatever element had the ID of 'section-header'

Semantics

Over the years, HTML, along with most languages, has evolved. Where we used to just use 'divs' (<div></div>), there are now a variety of more descriptive elements we can use to break up our code:

<nav></nav> could be used for describing a navigation bar at the top of the screen.

<main></main> could be used for the section where the most of the information is displayed.

<section></section> could be used to split the main area up into groups of relevant information, usually has a header. This web page is an example, each section separated between two lines is a new section

<article></article> could be used to specify independant, self-contained text such as a blog post.

<footer></footer> could be used to specify where the footer is.

<aside></aside> could be used to specify where a side nav bar lives, or a column at the side where ads may be shown.

Class and ID

You can assign classes and IDs to any HTML element. They are useful to help identify an element or group of elements, so that you can apply specific rules to them later on.
