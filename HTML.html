Text Formatting

Bold Text

You can make text bold by adding a pair of <strong></strong> tags around the desired text.

Example:

He was this close to scoring a goal!

Itallics

You can make text itallic by adding a pair of <em></em> tags around the desired text.

Example:

He was this close to scoring a goal!

Underline

You can make text underlined by adding a pair of <u></u> tags around the desired text.

Example:

He was this close to scoring a goal!

Strikethrough

You can make text have a strikethrough by adding a pair of <s></s> tags around the desired text.

He was this close wasn't even close scoring a goal!

Headers

Header tags can be used to create text in various preset sizes, and come with a number of formatting options already applied to them. Use tags <h1></h1> to <h6></h6> to get the size you want.

Example:

This is h1 text

This is h2 text

This is h3 text

This is h4 text

This is h5 text
This is h6 text
